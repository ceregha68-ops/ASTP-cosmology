# ASTP-cosmology
Код и модели для космологической теории ASTP (альтернатива ΛCDM без тёмной энергии)

**ASTP (Anisotropic Space‑Time Propagation)** – космологическая модель, в которой ускоренное расширение Вселенной объясняется не тёмной энергией, а геометрическим эффектом (параметр `x_eff ≈ 0.41`). Пространство – замкнутая 3‑сфера, расширяющаяся линейно.

## Основные публикации на Zenodo

- **Основная статья** (феноменологическая модель, подгонка к Pantheon+):  
  [DOI: 10.5281/zenodo.19522052](https://doi.org/10.5281/zenodo.19522052)
- **Математическое приложение** (строгий вывод формул):  
  [DOI: 10.5281/zenodo.19522462](https://doi.org/10.5281/zenodo.19522462)

## Содержимое репозитория

- `astp_pantheon_fit.py` – код для подгонки параметров `H₀` и `x_eff` к сверхновым Pantheon+.
- `astp_growth.py` – код для анализа роста структур (уравнение с параметром `β`).
- `plots/` – папка с графиками (будет создана при запуске).
- `requirements.txt` – список необходимых библиотек Python.
